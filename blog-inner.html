Kegiatan Sehari-hari Sebagai Mahasiswa Tingkat Akhir

Posted 1 Desember 2021

Hello!, Perkenalkan nama saya Muhammad Naufal Abiyyu, biasa dipanggil Nopal. Saat ini saya berusia 21 tahun dan sedang menjelani perkuliahan jurusan sistem informasi tingkat akhir. Kali ini saya akan menceritakan keseharian saya sebagai seorang mahasiswa tingkat akhir.

Keseharian saya saat ini aktif mengikuti pembelajaran didalam maupun diluar perkuliahan. Keseharian saya didalam perkuliahan seperti mengikuti perkuliahan tatap muka setiap hari, mengerjakan tugas dan memikirkan ide untuk skripsi, seperti halnya dengan mahasiswa tingkat akhir lainnya. Sedangkan keseharian saya diluar perkuliahan saat ini saya sedang mengikuti program magang dari MBKM, program tersebut diselenggarakan oleh Kemendikbud untuk memberikan kesempatan kepada mahasiswa belajar dan mengembangkan diri melalui aktivitas di luar kelas perkuliahan. Pada program magang ini saya mengambil profesi sebagai UI/UX Designer pada sebuah perusahaan PT. Impactbytle Teknologi Edukasi,

UI/UX Designer itu sendiri bertugas mengurus User Interface, dan User Experience dengan memaksimalkan pengalaman pengguna dalam menggunakan suatu aplikasi atau website.

Kegiatan magang tersebut dilaksanakan setiap hari senin sampai minggu selama 4 bulan dari agustus sampai desember 2021 dan Mahasiswa diharuskan membuat dan melaporkan laporan kegiatan harian (log book) selama kegiatan magang berlangsung pada setiap hari kerja.

Selain sedang menjalani magang, saat ini saya sedang menjalankan project bersama tim dengan membuat sebuah mobile application bernama AJIB (Antar Jemput Izin Bermotor), Aplikasi tersebut nantinya dapat melayani berbagai pengurusan izin di jakarta dengan lebih mudah dan cepat.

AJIB merupakan Layanan Antar Jemput Izin Bermotor milik Dinas Penanaman Modal dan Pelayanan Terpadu Satu Pintu (PM dan PTSP) DKI Jakarta menjadi salah satu inovasi layanan yang diminati warga.

Aplikasi tersebut nantinya dapat melayani berbagai pengurusan izin di jakarta dengan lebih mudah dan cepat, dengan begitu warga jakarta tidak perlu mendatangi service point Unit Pelaksana PM dan PTSP dalam mengajukan permohonan izin dan nonizin, Pemohon cukup memesan layanan AJIB melalui aplikasi.
